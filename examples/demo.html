📖 使用方法

直接引用 CSS 文件

<!-- 在 HTML head 中引用 -->
<link rel="stylesheet"
      href="https://cdn.jsdelivr.net/npm/@shingchil/webfont@latest/LxgwWenkaiGB-Regular.min.css">

<style>
body {
    font-family: 'LxgwWenkaiGB', serif;
}
</style>

🎯 支持的字体

  • LxgwWenkaiGB - 霞鹜文楷 GB (Regular)
  • SourceHanSerif - 思源宋体 (Regular, Bold)
  • ZhuqueFangsong - 朱雀仿宋 (Regular)
  • Chillkai - 寒蝉正楷 (Regular)
⚠️ 注意：字体切片基于 Google Noto Serif SC 的 unicode-range 方案，字体中不存在的字符会被自动跳过。
